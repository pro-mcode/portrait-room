## Overview

Portrait Room is a professional photography services website designed to showcase and book a variety of photo shoots including outdoor, commercial, wedding, and long-hour sessions. It features a clean, modern UI with responsive design and smooth navigation.

## Features

- Showcase multiple photography services
- Fully responsive layout
- Interactive galleries for each service
- Styled with TailwindCSS and SCSS
- Built with modular React components

## Tech Stack

Frontend:

- React
- JavaScript (ES6+)
- TailwindCSS
- SCSS

Other Tools:

- Git
- React Router
- Vite

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/pro-mcode/portrait-room.git
cd demo-travely
npm install
```

## Usage

```bash
npm run dev
npm run build
npm run preview
```
